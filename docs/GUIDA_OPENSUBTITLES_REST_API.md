# Guida OpenSubtitles - Transcriber Pro

## Introduzione

Transcriber Pro supporta l'upload automatico dei sottotitoli generati su **OpenSubtitles.org** tramite la **REST API** moderna.

### Funzionalità

- Upload automatico al termine dell'elaborazione
- Verifica automatica duplicati (tramite hash video) prima dell'upload
- Autenticazione moderna con API Key
- Configurazione completamente via GUI (nessun file da modificare manualmente)

---

## Prerequisiti

### 1. Account OpenSubtitles

- Registrazione gratuita: [opensubtitles.com](https://www.opensubtitles.com/en/newuser)
- Verifica email obbligatoria per attivare l'account
- Limite upload account gratuito: 10 sottotitoli/giorno

### 2. API Key

1. Vai su [opensubtitles.com/consumers](https://www.opensubtitles.com/consumers)
2. Login con il tuo account
3. Clicca **"Create Consumer"**
4. Compila il form (App Name: Transcriber Pro, Purpose: Personal use)
5. Copia la API Key generata (mostrata una sola volta — salvala subito)

---

## Configurazione

Tutte le credenziali si configurano in **Settings → OpenSubtitles**:

| Campo | Contenuto |
|-------|-----------|
| **Username** | Username o email OpenSubtitles |
| **Password** | Password del tuo account |
| **API Key** | La key ottenuta su opensubtitles.com/consumers |
| **Auto Upload** | Se abilitato, carica automaticamente al termine dell'elaborazione |

Nessun file JSON da creare manualmente — tutto viene salvato in `~/.transcriberpro/config.json`.

---

## Utilizzo

### Upload Automatico

1. Configura le credenziali in **Settings → OpenSubtitles**
2. Abilita **Auto Upload**
3. Elabora il video normalmente (Trascrizione → Traduzione)
4. Al termine, il sistema verifica automaticamente i duplicati e carica i sottotitoli

Log di conferma:
```
✅ Sottotitoli caricati su OpenSubtitles
   ID: 1234567890
   URL: https://www.opensubtitles.org/it/subtitles/1234567890
```

### Verifica Duplicati

Prima di ogni upload, il sistema:
1. Calcola l'hash del video (algoritmo OpenSubtitles)
2. Interroga l'API per trovare sottotitoli con lo stesso hash e lingua
3. Se già presenti → skip con log informativo
4. Se assenti → procede con l'upload

---

## Metadata Inviati

Transcriber Pro invia automaticamente metadata completi per ogni upload:

| Metadata | Fonte |
|----------|-------|
| IMDb ID | TMDB / OMDB (rilevato automaticamente) |
| Lingua | Rilevata automaticamente (formato ISO 639-2) |
| Release Name | Nome del file video |
| Video Hash | Calcolato localmente (per verifica duplicati) |
| Video Size | Dimensione file in byte |
| Comments | "Generated by Transcriber Pro" |

> **Importante:** per garantire l'IMDb ID, cerca i metadata TMDB prima di avviare l'upload. Rinomina i file video includendo l'anno (es: `Film Title (2024).mkv`) per facilitare il rilevamento.

## Troubleshooting

| Errore | Causa | Soluzione |
|--------|-------|-----------|
| Autenticazione fallita | Credenziali errate | Verifica username/password nelle Settings |
| API Key non valida | Key errata o scaduta | Rigenera su opensubtitles.com/consumers |
| Rate limit exceeded | Troppi upload | Attendi 24h |
| IMDb ID not found | Metadata TMDB mancanti | Cerca i metadata prima dell'upload |
| Duplicate subtitle | Sottotitoli già presenti | Comportamento normale — upload saltato |

---

## Sicurezza

- Le credenziali sono salvate localmente in `~/.transcriberpro/config.json`
- Non vengono mai trasmesse a server terzi (solo a OpenSubtitles)
- Non vengono mai loggate in chiaro
- Il file `config.json` è escluso dal repository tramite `.gitignore`
